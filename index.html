Endpoints

Projects

Base URL:`https://api.AppName.com/v1/projects`

  • GET /projects: Retrieve a list of all projects
  • POST /projects: Create a new project. Required parameters: `name`, `description`.
  • GET /projects/{project_id}: Retrieve details of a specified project
  • PUT /projects/{project_id}: Update an existing project
  • DELETE /projects/{project_id}: Delete a project

Files

  • GET /files: Retrieve a list of all projects
  • POST /files: Create a new file. Required parameters: `name`, `description`.
  • GET /files/{file_id}: Retrieve details of a specified file
  • PUT /files/{file_id}: Update an existing file
  • DELETE /files/{file_id}: Delete a file

Request and Response Formats

All API requests and response are formatted in JSON. Here's an example of a request to create a new project:

Post /v1/projects { "name": "new Project", "description": "A brief description of the project"}
